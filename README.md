# Explore With Me

Приложение для публикации событий и поиска компании для участия в них.

---

## Архитектура

Проект состоит из двух сервисов:

- **main-service** — основной сервис приложения
- **stats-service** — сервис сбора и агрегации статистики просмотров

Каждый сервис:
- запускается в отдельном Docker-контейнере;
- использует собственную базу данных PostgreSQL;
- взаимодействует с другими сервисами по HTTP.

---

## Основной функционал

### Main Service
- создание, редактирование и публикация событий;
- подача заявок на участие в событиях;
- подтверждение и отклонение заявок инициатором события;
- публичный поиск и фильтрация событий;
- управление категориями, пользователями и подборками (admin API).

### Stats Service
- фиксация обращений к публичным эндпоинтам;
- получение статистики просмотров за период;
- поддержка подсчёта уникальных просмотров.

---

## Дополнительная фича

### Комментирование событий

Реализована возможность работы с комментариями к событиям:
- авторизованные пользователи могут оставлять комментарии к опубликованным событиям;
- администратор может модерировать комментарии;

---

## Интеграция со статистикой

- каждый запрос к публичным эндпоинтам фиксируется в сервисе статистики;
- при получении событий данные о количестве просмотров подтягиваются из stats-service;
- агрегация статистики выполняется на уровне базы данных.
